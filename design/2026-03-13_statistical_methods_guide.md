# 통계분석 기법 개념 정리

- 작성일: 2026-03-13
- 적용 맥락: AI 출판 설문 연구

## 1. 기술통계

### 개념

기술통계는 자료의 전체 모습을 요약해서 보여주는 분석이다. 평균, 표준편차, 빈도, 비율, 최솟값, 최댓값 같은 값을 통해 표본의 기본 특성을 파악한다.

### 언제 쓰는가

- 응답자 구성이 어떤지 보고 싶을 때
- 주요 변수의 전반적 수준을 먼저 확인할 때
- 본격적인 가설검정 전에 데이터 상태를 점검할 때

### 이 연구에서의 예

- `P_score` 평균이 어느 정도인지
- `U1`에서 AI를 자주 쓰는 사람이 얼마나 되는지
- `T_pre`와 `T_post`가 어떤 방향으로 움직였는지

### 보고 항목

- 연속형: 평균, 표준편차, 최소값, 최대값, 왜도, 첨도
- 범주형: 빈도, 백분율

## 2. 독립표본 t-검정

### 개념

독립표본 `t-test`는 서로 다른 두 집단의 평균 차이가 우연인지 아닌지를 검정하는 방법이다. 두 집단은 서로 독립적이어야 한다.

### 언제 쓰는가

- 집단이 2개일 때
- 종속변수가 연속형일 때
- 두 집단 평균을 비교하고 싶을 때

### 이 연구에서의 예

- 도서관 종사자와 비종사자의 `P_score` 평균 차이
- AI 출판 관련 경험 유무에 따른 위험 인식 차이

### 기본 가설

- `H0`: 두 집단 평균은 같다.
- `H1`: 두 집단 평균은 다르다. 또는 한 집단이 더 높다.

### 전제

- 집단 간 독립성
- 종속변수의 대략적 정규성
- 등분산성

### 주요 보고값

- `t`, `df`, `p`
- 각 집단 평균과 표준편차
- 효과크기 Cohen's `d`

## 3. 대응표본 t-검정

### 개념

대응표본 `t-test`는 같은 사람을 두 번 측정했을 때 평균 차이가 있는지 검정하는 방법이다. 전후 비교, 반복 측정에 적합하다.

### 언제 쓰는가

- 동일 응답자의 사전·사후 점수를 비교할 때
- 짝지어진 두 측정값이 있을 때

### 이 연구에서의 예

- `T_pre`와 `T_post`의 차이 검정

### 기본 가설

- `H0`: 평균 차이는 0이다.
- `H1`: 평균 차이는 0이 아니다.

### 전제

- 차이점수의 대략적 정규성
- 동일 응답자 기준의 짝지어진 자료

### 주요 보고값

- 평균 차이
- `t`, `df`, `p`
- 효과크기 Cohen's `dz`

## 4. 단일표본 비율 z-검정

### 개념

단일표본 비율 `z-test`는 어떤 이분형 응답의 비율이 특정 기준값과 다른지를 검정하는 방법이다. 예를 들어 찬성 비율이 50%를 넘는지 볼 때 사용한다.

### 언제 쓰는가

- 결과가 `예/아니오`처럼 이분형일 때
- 비교 기준 비율이 명확할 때
- 표본 수가 충분히 클 때

### 이 연구에서의 예

- 표시 의무제 찬성 비율이 50%를 초과하는지 검정

### 기본 가설

- `H0`: 표본 비율은 기준 비율과 같다.
- `H1`: 표본 비율은 기준 비율보다 크거나 다르다.

### 전제

- 무작위 또는 준무작위 표집이 이상적
- `np0 >= 5`, `n(1-p0) >= 5`

### 주의

표본이 작으면 `z-test`보다 exact binomial test가 더 적절하다.

## 5. 일원분산분석 ANOVA

### 개념

ANOVA는 3개 이상 집단의 평균 차이를 한 번에 비교하는 방법이다. 집단이 많을 때 여러 번 `t-test`를 하면 1종 오류가 커지기 때문에 ANOVA를 쓴다.

### 언제 쓰는가

- 집단이 3개 이상일 때
- 종속변수가 연속형일 때
- 평균 차이 존재 여부를 먼저 확인하고 싶을 때

### 이 연구에서의 예

- AI 활용 빈도 5개 수준에 따라 `P_score`가 다른지 검정

### 기본 가설

- `H0`: 모든 집단 평균은 같다.
- `H1`: 적어도 하나의 집단 평균은 다르다.

### 전제

- 관측치 독립성
- 집단별 정규성
- 등분산성

### 주요 보고값

- `F`, `df`, `p`
- 집단별 평균과 표준편차
- 효과크기 `η²`
- 사후검정 결과

### 사후검정

ANOVA가 유의하면 어떤 집단끼리 차이가 있는지 Tukey HSD나 Bonferroni로 추가 확인한다.

## 6. 단순회귀분석

### 개념

단순회귀는 하나의 독립변수가 하나의 종속변수를 얼마나 예측하는지 보는 방법이다. 변수 간 방향성과 예측력을 함께 파악할 수 있다.

### 언제 쓰는가

- 독립변수 1개, 종속변수 1개일 때
- 연속형 변수 관계를 보고 싶을 때
- 영향 방향과 설명력을 확인하고 싶을 때

### 이 연구에서의 예

- `R_score`가 높을수록 `P_score`도 높아지는지 검정

### 기본 모형

`Y = β0 + β1X + ε`

### 기본 가설

- `H0`: `β1 = 0`
- `H1`: `β1 ≠ 0` 또는 `β1 > 0`

### 전제

- 선형성
- 오차의 독립성
- 등분산성
- 잔차의 대략적 정규성
- 극단적 이상치가 크지 않을 것

### 주요 보고값

- 회귀계수 `β`
- 표준오차 `SE`
- `t`, `p`
- `R²`
- `F`

## 7. 크론바흐 알파

### 개념

크론바흐 알파는 여러 문항이 하나의 척도로 묶일 수 있을 만큼 서로 일관되게 움직이는지 보는 신뢰도 지표다.

### 언제 쓰는가

- 다문항 척도를 평균 또는 합산 점수로 만들기 전에
- 척도 문항의 내적 일관성을 확인하고 싶을 때

### 이 연구에서의 예

- `L1~L4`를 하나의 이해도 척도로 묶을 수 있는지
- `R1~R4`를 하나의 위험 인식 척도로 묶을 수 있는지
- `P1~P6`를 하나의 제도 필요성 척도로 묶을 수 있는지

### 일반적 해석 기준

- `.90` 이상: 매우 높음
- `.80` 이상: 좋음
- `.70` 이상: 수용 가능
- `.60` 미만: 재검토 필요

### 주의

- 알파가 높다고 반드시 단일 차원 척도라는 뜻은 아니다.
- 문항 수가 많으면 알파가 다소 높아질 수 있다.
- 문항-총점 상관과 내용 타당도도 함께 봐야 한다.
